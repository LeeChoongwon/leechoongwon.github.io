## 풀이
1. i-1칸까지 채운 뒤, 1X2 타일을 하나 세로로 채우는 경우
2. i-2칸까지 채운 뒤, 2X1 타일을 위 아래로(가로) 2개 채우는 경우

두 경우의 수의 합이 i칸까지 채우는 방법이 된다. <br />

`d[i] = d[i-1] + d[i-2]`

## 코드
파이썬 (python)
```py
n = int(input())
d = [0,1,2]

for i in range(3,n+1):
    d.append(d[i-1] + d[i-2])

print(d[n] % 10007)
```

문제를 대충 읽고, 10007로 나눈 나머지를 구하지않아 틀렸었다. `문제를 끝까지 읽자.`
